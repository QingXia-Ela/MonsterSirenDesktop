## 插件内登录

登录的话使用的是手机号 + 验证码登录，没有测试国外登录环境，自己随缘吧（逃

在登录页面按照操作完成登录后，假如看到提示：

```
登录成功，可以关闭本页面开始听音乐了
```

那就证明现在有权限访问歌单了！

还没完，先别关闭软件，你现在需要设置自己的 uid。

## 设置你的网易云 uid

使用网页版浏览器，进入自己的个人主页，此时看一下浏览器的地址栏，可以注意到最后那里的 `id=xxxx`，其中 `xxxx` 就是你的 uid。

![](../../../assets/插件/1.png)

记住这串数字，然后访问以下 URL： http://localhost:53753/uid/set?uid=xxxx ，其中，把 `xxxx` 替换成你的 uid。

如上图，我的 uid 是 `559691011` ，那么 URL 就是 `http://localhost:53753/uid/set?uid=559691011`

假如访问页面看到类似下面的提示（提示 message 为 ok）：

```json
{"message":"ok"}
```

那就证明你已经完成了所有操作，可以开始听网易云的音乐了！

:::note
不要尝试测试软件的健壮性与稳定性，问就是没搞。
:::
