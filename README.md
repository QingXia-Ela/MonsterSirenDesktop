# Monster Siren Desktop

塞壬唱片桌面版 + 功能增强

**注：这只是一个同人作品，不代表官方任何内容**

项目还处于起步阶段，如果有加入意愿或者建议可以开个 issue 讨论

预计完成的功能如下：

- 一些页面基本操作自定义
- 背景图切换
- 能够读取本地歌曲进行播放
- 能够下载网站内的音频并进行格式转换
- 桌面歌词
- 具备接入第三方音乐平台的机制
- 具备离线访问功能
